# 📦 Capstone Project - E-Commerce Platform

This repository contains the source code for the **E-Commerce Platform** built using:

- 🖥️ **Frontend:** React.js
- ⚙️ **Backend:** .NET Core API
- 🗂️ **Database:** SQL Server
